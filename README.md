# Navissos Pet Shop - Frontend

This is the **Navissos Pet Shop** frontend application built using **Angular**. The project serves as the user-facing interface for managing pet listings, browsing pets, and interacting with other features. It integrates with a backend API (Spring Boot) for data persistence.

---

## 🚀 Features

- View a list of pets with pagination.
- Browse detailed pet profiles.
- Home page with recommended pets.
- About page with the shop's history.
- Optimized for performance and responsiveness.

---

## 🛠️ Installation and Setup

### Prerequisites

1. [Node.js](https://nodejs.org) (>= 16.x)
2. [Angular CLI](https://angular.io/cli) (>= 15.x)
3. A running instance of the backend service (check the backend repository).

### Steps to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/navissos-pet-shop-frontend.git
   cd navissos-pet-shop-frontend
2. Install dependencies:
   ```bash
   npm install
3. Configure API endpoints
4. Run the development server:
   ```bash
    ng serve
  The app will be available at http://localhost:4200
